# Language Translator

 A simple **Python Tkinter Language Translator** using **Deep-Translator API**.  

Users can enter text, select a source language and target language, and get instant translations.  

## Features
- Input box for text  
- Source language selection (English, Telugu, Hindi, Spanish)  
- Target language selection  
- Translate button  
- Translated text display  

## Tech Stack
- Python 3.x  
- Tkinter (for GUI)  
- Deep-Translator (for translations)  

## How to Run
1. Clone the repository or download `final_translator.py`  
2. Install dependencies:  
   pip install deep-translator
